# v4.2 升级日志

## 升级日期
2026-04-04

## 新增功能

### 1. 爆款视频本地下载 (Competitor-Viral-Analyst)
- **文件**: `skills/competitor-viral-analyst/scripts/build_hotspot_library.py`
- 新增 `process_viral_video()` 函数
- 保存到 `knowledge_base/shared_library/trending_videos/{platform}/`
- 支持 yt-dlp 和 requests 两种下载方式
- 保存视频元数据到 `metadata.json`
- 如无法下载则保存缩略图

### 2. 统一共享资源库 (Shared Resource Library)
- **新增目录**: `knowledge_base/shared_library/`
- 目录结构:
  ```
  shared_library/
  ├── trending_videos/{platform}/  # 爆款视频
  ├── hotspots/                    # 热词库
  ├── script_templates/            # 脚本模板(Markdown)
  ├── product_anchors/           # 产品视觉锚点
  ├── sample_clips/              # 示例片段
  └── README.md                    # 说明文档
  ```

### 3. 新部署自动初始化 (Workspace-Initializer)
- **文件**: `skills/workspace-initializer/SKILL.md`
- **脚本**: `skills/workspace-initializer/scripts/init_workspace.py`
- 检测首次运行 (`~/.openclaw/init_done`)
- 自动创建目录结构
- 检查 FFmpeg 依赖
- 从 GitHub 同步基础资源

### 4. 本地资源服务器
- **文件**: `skills/workspace-initializer/scripts/start_resource_server.py`
- 在 `localhost:8080` 提供 Web 界面
- 支持浏览和下载视频、热词、脚本
- 仅监听 127.0.0.1

## 使用方式

```bash
# 1. 初始化（首次运行自动执行）
python skills/workspace-initializer/scripts/init_workspace.py

# 2. 运行主流程
python skills/script-writer/scripts/run_v4.py "椰子水" 55

# 3. 启动资源服务器
python skills/workspace-initializer/scripts/start_resource_server.py --port 8080

# 访问 http://localhost:8080
```
