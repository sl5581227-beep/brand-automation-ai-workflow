# v4.5 升级日志

## 升级日期
2026-04-04

## 升级目标
使用产品标准图 + 文本共同指导即梦AI生成片段，大幅提升外观一致性

## 核心变更

### 1. 产品标准图支持 (Clip-Generator)
- **文件**: `skills/clip-generator/scripts/generate_clip.py`
- 生成前从 `products_manifest.json` 获取产品标准图
- 使用即梦AI图生视频API（image-to-video）替代纯文本生成
- 支持多张标准图（front, angle_45, side等）
- 失败时自动尝试其他角度的标准图

### 2. 提示词过滤
- 过滤prompt中的外观描述词，只保留动作和环境
- 示例：
  - 旧: "close-up of white cylindrical bottle, clear liquid..."
  - 新: "slow push-in on the product, natural window light..."

### 3. 外观核验阈值调整
- **文件**: `skills/product-appearance-check/scripts/check_product_appearance.py`
- 阈值从75%调整到80%
- 禁止特征零容忍
- 新增"宽容机制"：初级失败但高级通过可给予一次机会

### 4. 产品标准图提取
- **文件**: `skills/clip-generator/scripts/extract_product_images.py`
- 从素材01提取产品图片
- 更新products_manifest.json中的standard_images字段

## API 变更

### 即梦AI调用
```python
# 旧：纯文本生成
generate_with_dreamina_text_to_video(prompt)

# 新：图生视频
generate_with_dreamina_image_to_video(standard_image, prompt)
```
