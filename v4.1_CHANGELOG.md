# v4.1 升级日志

## 升级日期
2026-04-04

## 新增功能

### 1. 自动化产品发现与注册 (Product-Scanner)
- **文件**: `skills/product-scanner/SKILL.md`, `skills/product-scanner/scripts/scan_products.py`
- 扫描 `C:\Users\Administrator\Desktop\素材01\` 目录
- 自动识别 PPTX/PDF 中的产品名称
- 生成/更新 `knowledge_base/products_manifest.json`

### 2. 智能重试与降级策略 (Clip-Generator)
- **文件**: `skills/clip-generator/scripts/generate_clip.py`
- 调用即梦AI失败时自动重试（最多3次）
- 每次重试修改提示词（缩短/改变镜头类型）
- 3次都失败后记录到 `logs/failed_clips.json`，跳过该片段继续下一个
- 不中断整个流程

### 3. 片段缓存与复用 (Clip-Cache)
- **文件**: `skills/clip-generator/scripts/clip_cache.py`
- 维护 `knowledge_base/clip_cache.json`
- 基于场景描述MD5哈希查重
- 缓存有效期30天
- 评分≥75分才复用

### 4. 成本统计与配额管理 (Cost-Tracker)
- **文件**: `skills/cost_manager/SKILL.md`, `skills/cost_manager/scripts/cost_tracker.py`
- 每次调用记录：时间戳、产品ID、时长、费用
- 每日汇总 `logs/cost_daily.json`
- 每周汇总 `logs/cost_weekly.json`
- 配额阈值警告（默认每日100元，每周500元）

### 5. 异步任务队列
- **文件**: `skills/script-writer/scripts/run_v4.py`
- 支持 `--task_file=` 参数读取JSON任务列表
- 支持 `--concurrency=` 参数（建议保持1）
- 任务失败不影响下一个

### 6. 多平台输出适配 (Multi-Ratio)
- **文件**: `skills/video-editor/scripts/multi_ratio.py`
- 自动生成三个比例版本：
  - `原名称_9x16.mp4` (1080x1920)
  - `原名称_1x1.mp4` (1080x1080)
  - `原名称_16x9.mp4` (1920x1080)

### 7. 成片元数据追踪 (Final-DB)
- **文件**: `skills/final-qc/scripts/evaluate_final.py`, `skills/final-qc/scripts/query_videos.py`
- 生成唯一UUID
- 记录到 `metadata/final_db.json`
- 支持按产品/日期/评分查询

## 使用的命令

```bash
# 单任务
python run_v4.py --product=qingshang_coconut_water --duration=55

# 批量任务
python run_v4.py --task_file=tasks.json

# 查询视频
python skills/final-qc/scripts/query_videos.py all
python skills/final-qc/scripts/query_videos.py product qingshang_coconut_water

# 扫描产品
python skills/product-scanner/scripts/scan_products.py

# 查看成本
python skills/cost_manager/scripts/cost_tracker.py
```
