## Instructions

1. **提取关键词**
   - 从用户输入中提取产品/主题关键词
   - 如未提供，调用 Brand-Analyzer 获取产品核心卖点作为关键词

2. **执行搜索**
   - 使用 `web_search` 工具搜索：
     - `{关键词} + 爆款视频`
     - `{关键词} + 热门话题`
     - `{关键词} + 高赞视频`
   - 搜索平台：抖音、小红书、TikTok相关关键词

3. **整理结果**
   - 提取标题、点赞数、话题标签
   - 按热度/相关度排序
   - 输出前5个最相关的热点趋势

4. **输出格式**
   ```
   🔥 热点趋势检索（关键词："XXX"）
   
   1. [趋势标题]
      平台：抖音/小红书
      点赞：10w+
      话题标签：#xxx #yyy
      链接：https://...
   
   2. ...
   ```

## 热点趋势分类

| 类型 | 说明 |
|------|------|
| 场景型 | 运动后、早餐、办公室等使用场景 |
| 功能型 | 肠道健康、解渴、清爽等功效 |
| 人群型 | 健身人群、上班族、学生等 |
| 节日型 | 夏季限定、节假日等 |
